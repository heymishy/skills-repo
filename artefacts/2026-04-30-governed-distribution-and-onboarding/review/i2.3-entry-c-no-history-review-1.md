## FINDINGS

### 1-L1 — LOW — AC2 missing WHEN clause; Given/When/Then structure incomplete

AC2 states: "Given /orient has detected Entry C, then the routing output distinguishes Entry C from Entry B in the signal description: e.g., 'No application logic files found in the expected source directories (src/, app/, lib/); Entry B (code-first) is not applicable. Treating this as Entry C (no-history brownfield).' — so the operator understands why Entry C was selected over Entry B." There is no WHEN clause. As with the parallel issue in i2.2 AC2, the implicit When is "when /orient produces its routing output," but the omission leaves the AC structurally incomplete per GWT format requirements.

**Recommended action:** Add a WHEN clause — e.g. "Given /orient has detected Entry C, **when /orient produces its routing output**, then the output distinguishes Entry C from Entry B in the signal description...". This makes AC2 independently exercisable.
