### Task 3 — Wire TENANT guard into 9 read handlers (AC1-AC9)

For each handler below, REPLACE the existing `if (!journey) { return 404 }` check with the guard call. The guard already handles null journey → NOT_FOUND → 404.

**Pattern:**
```js
// REMOVE the existing null check:
// if (!journey) { res.writeHead(404, ...); res.end(...); return; }

// REPLACE WITH:
try { requireJourneyAccess(journey, req.session, POLICY.TENANT); }
catch (err) {
  res.writeHead(asHttpResponse(err, POLICY.TENANT), { 'Content-Type': '<mime>' });
  res.end(JSON.stringify({ error: 'Not found' }));
  return;
}
```

**Special case — handleGetJourney (line 239, AC2):**
Does NOT look up a journey by ID. Add conditional journeyId guard AFTER the auth redirect check:
```js
// Guard specific journey if journeyId param is present (per-journey view)
var journeyId = req.params && req.params.journeyId;
if (journeyId) {
  var _guardJourney = _journeyStore.getJourney(journeyId);
  try { requireJourneyAccess(_guardJourney, req.session, POLICY.TENANT); }
  catch (err) {
    res.writeHead(asHttpResponse(err, POLICY.TENANT), { 'Content-Type': 'text/html; charset=utf-8' });
    res.end(renderShell({ title: 'Not found', bodyContent: '<p>Not found.</p>', user: { login: req.session.login || '' } }));
    return;
  }
}
// Existing list-page logic follows unchanged...
```
