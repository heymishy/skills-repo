**D1 — Session write sync deviation (minor):** `writeSession` calls `fs.writeFileSync` not `fs.promises.writeFile`. For the current solo-developer / single-tenant deployment profile this is benign; for a production multi-tenant deployment this should be converted to async I/O. Flagged as a follow-up candidate; not blocking release.
