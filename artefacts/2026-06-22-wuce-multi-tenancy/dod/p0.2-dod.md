## DoD Observations

1. **check-skill-contracts.js path bug:** The pre-commit hook was broken since migration `7ea39aa` (moved SKILL.md files but didn't update script). This silently blocked all post-migration commits from any feature branch. Fixed in this PR as a prerequisite. **Improve candidate:** add a CI step that runs `check-skill-contracts.js` on push to catch this class of hook breakage before it blocks developer commits.

2. **AC2/handleGetJourney design note:** The route handler name `handleGetJourney` is misleading — it handles the journey list page, not a per-journey view. The per-journey HTML view is `handleGetJourneyById`. This caused the AC2 discrepancy. **Improve candidate:** add a handler-to-route mapping table to the feature's reference artefact so future story ACs can reference routes rather than handler names.
