## AC7 — featureSlug collision guard: same slug, different tenants, separate directories

**Covers:** AC7

**Steps:**
1. Set `WUCE_TENANT_ROOT_BASE = $TMPDIR_TEST`.
2. Compute `getRepoRoot(req)` for `{ tenantId: 'org-a' }` → call it `rootA`.
3. Compute `getRepoRoot(req)` for `{ tenantId: 'org-b' }` → call it `rootB`.
4. Assert `rootA !== rootB`.
5. Write a file at `path.join(rootA, 'artefacts', 'my-feature', 'test.txt')` with content `'org-a'`.
6. Write a file at `path.join(rootB, 'artefacts', 'my-feature', 'test.txt')` with content `'org-b'`.
7. Assert reading `rootA` path returns `'org-a'`; reading `rootB` path returns `'org-b'`.

**Expected result:** Identical featureSlugs coexist under separate tenant roots without collision.
