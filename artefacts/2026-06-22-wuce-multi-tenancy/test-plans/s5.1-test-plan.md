## NFR Tests

### NFR-SEC-1 — check-beta-health.sh does not log credential values

- **NFR addressed:** Security
- **Measurement method:** Read scripts/check-beta-health.sh; confirm no echo/print of $UPSTASH_REDIS_REST_TOKEN, $DATABASE_URL, or equivalent
- **Pass threshold:** Zero occurrences of credential variable values in script output
- **Tool:** Code inspection (automated grep)

### NFR-PERF-1 — GET /health check completes within 10 seconds

- **NFR addressed:** Performance
- **Measurement method:** check-beta-health.sh uses `curl --max-time 10`
- **Pass threshold:** curl exits within 10 seconds; no timeout error
- **Tool:** check-beta-health.sh (automated)

### NFR-PORT-1 — check-beta-health.sh runs on bash 3+

- **NFR addressed:** Portability
- **Measurement method:** Script uses only POSIX-compatible constructs; no bash 4+ features (`declare -A`, `mapfile`, etc.)
- **Pass threshold:** Script runs without syntax errors on bash 3.2 (macOS default) and bash 5 (Linux)
- **Tool:** `bash --version` + code inspection (manual)
