## Unit Tests

### requireJourneyAccess — legacy journey (ownerId null) passes without error

- **Verifies:** AC1
- **Precondition:** Journey object with `ownerId: null`; session with `{ accessToken: 'tok', userId: '1', login: 'alice' }`
- **Action:** Call `requireJourneyAccess({ ownerId: null }, session, POLICY.TENANT)`
- **Expected result:** No error thrown; function returns without throwing
- **Edge case:** No

### requireJourneyAccess — owner access passes without error

- **Verifies:** AC2
- **Precondition:** Journey `{ ownerId: 'alice' }`, session `{ login: 'alice', accessToken: 'tok', userId: '1' }`
- **Action:** Call `requireJourneyAccess(journey, session, POLICY.TENANT)`
- **Expected result:** No error thrown
- **Edge case:** No

### requireJourneyAccess — non-owner throws FORBIDDEN

- **Verifies:** AC3
- **Precondition:** Journey `{ ownerId: 'bob' }`, session `{ login: 'alice', accessToken: 'tok', userId: '1' }`
- **Action:** `assert.throws(() => requireJourneyAccess(journey, session, POLICY.TENANT))`
- **Expected result:** Thrown error has `code === 'FORBIDDEN'`
- **Edge case:** No

### requireJourneyAccess — no accessToken throws UNAUTHENTICATED

- **Verifies:** AC4
- **Precondition:** Journey `{ ownerId: 'alice' }`, session `{ login: 'alice', userId: '1' }` (no accessToken)
- **Action:** `assert.throws(() => requireJourneyAccess(journey, session, POLICY.TENANT))`
- **Expected result:** Thrown error has `code === 'UNAUTHENTICATED'`
- **Edge case:** No

### requireJourneyAccess — null journey throws NOT_FOUND

- **Verifies:** AC5
- **Precondition:** `journey = null`, session `{ accessToken: 'tok', userId: '1', login: 'alice' }`
- **Action:** `assert.throws(() => requireJourneyAccess(null, session, POLICY.TENANT))`
- **Expected result:** Thrown error has `code === 'NOT_FOUND'`
- **Edge case:** Also test with `journey = undefined`

### asHttpResponse — TENANT policy maps FORBIDDEN to 404

- **Verifies:** AC6
- **Precondition:** Error object with `code === 'FORBIDDEN'`, policy = `POLICY.TENANT`
- **Action:** Call `asHttpResponse(err, POLICY.TENANT)`
- **Expected result:** Returns `{ status: 404 }` (or equivalent — the 404 response object)
- **Edge case:** No

### asHttpResponse — OWNER policy maps FORBIDDEN to 403

- **Verifies:** AC7
- **Precondition:** Error object with `code === 'FORBIDDEN'`, policy = `POLICY.OWNER`
- **Action:** Call `asHttpResponse(err, POLICY.OWNER)`
- **Expected result:** Returns `{ status: 403 }`
- **Edge case:** No

### asHttpResponse — UNAUTHENTICATED maps to 401 under TENANT policy

- **Verifies:** AC8
- **Precondition:** Error with `code === 'UNAUTHENTICATED'`, policy = `POLICY.TENANT`
- **Action:** Call `asHttpResponse(err, POLICY.TENANT)`
- **Expected result:** Returns `{ status: 401 }`
- **Edge case:** Also assert same result under OWNER policy — must always be 401

### asHttpResponse — NOT_FOUND maps to 404 under both policies

- **Verifies:** AC9
- **Precondition:** Error with `code === 'NOT_FOUND'`
- **Action:** Call `asHttpResponse(err, POLICY.TENANT)` and `asHttpResponse(err, POLICY.OWNER)`
- **Expected result:** Both return `{ status: 404 }`
- **Edge case:** No

### isSameTenant — returns true when both session and journey have no tenantId

- **Verifies:** AC10
- **Precondition:** `journey = {}`, `session = {}`
- **Action:** `assert.strictEqual(isSameTenant(journey, session), true)`
- **Expected result:** `true`
- **Edge case:** No

### isSameTenant — returns true when session.tenantId is undefined

- **Verifies:** AC10
- **Precondition:** `journey = { tenantId: 'org-a' }`, `session = { tenantId: undefined }`
- **Action:** `assert.strictEqual(isSameTenant(journey, session), true)`
- **Expected result:** `true` (Phase 0 passthrough — if either side is absent, return true)
- **Edge case:** Yes — this is the Phase 0 compatibility edge case

### Module loads without side effects

- **Verifies:** AC11
- **Precondition:** Fresh require of the module path
- **Action:** `const m = require('src/web-ui/middleware/journey-access'); assert.ok(typeof m.requireJourneyAccess === 'function'); assert.ok(typeof m.asHttpResponse === 'function'); assert.ok(typeof m.isSameTenant === 'function'); assert.ok(m.POLICY)`
- **Expected result:** All named exports present; no error on import; no network or disk I/O triggered
- **Edge case:** No

## NFR Tests

### requireJourneyAccess is synchronous (no I/O)

- **NFR addressed:** Performance — "Key construction is synchronous — no I/O, negligible latency impact" (story NFR)
- **Measurement method:** Assert no promise is returned; wrap in synchronous timing assert: the function must complete within a synchronous call stack (no `await` needed)
- **Pass threshold:** Function completes synchronously — return value is not a Promise
- **Tool:** Plain Node.js `assert` — `assert.ok(!(requireJourneyAccess(...) instanceof Promise))`
