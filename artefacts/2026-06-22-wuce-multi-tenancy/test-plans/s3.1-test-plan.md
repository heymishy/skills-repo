## Integration Tests

> **TDD note:** These tests are implemented by `scripts/smoke-test-pg.js` (already exists). They fail (or skip gracefully) right now because `DATABASE_URL` is not set. Once Neon is provisioned and `DATABASE_URL` is set, they pass. The "failing" state is `SKIP: DATABASE_URL not set`.

### T1 — migrate-schema-pg exits 0 and prints schema confirmation message

- **Verifies:** AC2
- **Components involved:** scripts/migrate-schema-pg.js → Neon Postgres via pg.Pool
- **Precondition:** `DATABASE_URL` set to a valid Neon connection string
- **Action:** `DATABASE_URL=<url> node scripts/migrate-schema-pg.js`
- **Expected result:** Exit code 0; stdout contains "Schema created (or already exists): journeys table + tenant_id index"

### T2 — INSERT test journey row

- **Verifies:** AC4 (insert leg)
- **Components involved:** smoke-test-pg.js → pg.Pool → Neon journeys table
- **Precondition:** `DATABASE_URL` set; schema migrated (T1)
- **Action:** `DATABASE_URL=<url> node scripts/smoke-test-pg.js`
- **Expected result:** stdout contains "INSERT ... OK"; no thrown error

### T3 — SELECT by journey_id returns the inserted row

- **Verifies:** AC4 (read leg)
- **Components involved:** smoke-test-pg.js → SELECT WHERE journey_id
- **Precondition:** T2 completed
- **Action:** Continues in same script run
- **Expected result:** stdout contains "SELECT ... OK (tenant_id=smoke-tenant)"

### T4 — accessToken absent from stored JSONB data

- **Verifies:** AC4 (strip-before-write invariant)
- **Components involved:** smoke-test-pg.js → data column JSONB
- **Precondition:** T3 completed
- **Action:** Reads `data` column from returned row; checks `stored.accessToken`
- **Expected result:** stdout contains "accessToken absent from stored data ... OK"; `stored.accessToken === undefined`

### T5 — DELETE removes the test row

- **Verifies:** AC4 (cleanup leg)
- **Components involved:** smoke-test-pg.js → DELETE WHERE journey_id
- **Precondition:** T2 completed
- **Action:** Continues in same script run
- **Expected result:** stdout contains "DELETE ... OK"; follow-up SELECT returns 0 rows; script exits 0; stdout contains "Neon smoke test PASSED"
