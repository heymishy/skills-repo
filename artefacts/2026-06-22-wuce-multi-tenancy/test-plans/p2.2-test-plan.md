### Filesystem isolation — org-a directory contains no org-b files

- **Verifies:** AC3
- **Components involved:** Filesystem; `getRepoRoot`; artefact-writing handlers
- **Precondition:** Both tenants have written artefact files under their respective directories in `tmpDir`
- **Action:** `const files = fs.readdirSync(path.join(tmpDir, 'org-a'), { recursive: true }); assert.ok(files.every(f => !f.includes('org-b')))`
- **Expected result:** Zero org-b paths in org-a directory tree (and vice versa)
- **Edge case:** Yes — the key storage isolation assertion

### Session store isolation — cross-namespace lookup returns nothing

- **Verifies:** AC4
- **Components involved:** Session-store adapter
- **Precondition:** `SESSION_STORE_PATH/org-a/sess-1.json` exists; `SESSION_STORE_PATH/org-b/sess-1.json` exists with different content
- **Action:** Read session with `{ tenantId: 'org-b', sessionId: 'sess-1' }` from session store; assert it does NOT return alice's session data
- **Expected result:** Returns bob's session data (or null for a session ID that only exists under org-a); never returns a cross-tenant session
- **Edge case:** Yes — cross-namespace read safety

### Same-tenant access — alice reading alice's own journey succeeds on all routes

- **Verifies:** AC5
- **Components involved:** All 9 read handlers + 2 mutating handlers, guard
- **Precondition:** alice's journey; alice's 4-field session
- **Action:** Call each handler with alice's session for alice's journey
- **Expected result:** No 4xx response on any handler — same-tenant access is not broken by Phase 2 changes

## NFR Tests

### Test suite completes in under 10 seconds

- **NFR addressed:** Performance — "test suite must complete in under 10 seconds"
- **Measurement method:** `time node tests/check-p2.2-tenant-isolation.js`
- **Pass threshold:** < 10000ms wall time
- **Tool:** Shell timing + explicit assertion in test: `assert.ok(endTime - startTime < 10000)`

### Temporary directories cleaned up after test run

- **NFR addressed:** Correctness — "test suite must clean up any temporary directories it creates"
- **Measurement method:** Assert `!fs.existsSync(tmpDir)` after test suite completes (in `after()` cleanup block)
- **Pass threshold:** tmpDir does not exist after test run completes (even if tests fail — use `try/finally`)
- **Tool:** Plain `assert` + `fs.rmSync(tmpDir, { recursive: true })`
