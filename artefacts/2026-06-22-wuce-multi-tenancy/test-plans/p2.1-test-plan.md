## Unit Tests

### getRepoRoot — returns tenant-scoped path when WUCE_TENANT_ROOT_BASE is set

- **Verifies:** AC1
- **Precondition:** `process.env.WUCE_TENANT_ROOT_BASE = '/data/tenants'`; session `{ tenantId: 'my-org' }`
- **Action:** `const root = getRepoRoot(makeReq({ session: { tenantId: 'my-org' } })); assert.strictEqual(root, path.resolve('/data/tenants/my-org'));`
- **Expected result:** `path.resolve('/data/tenants/my-org')` (platform-normalised absolute path)
- **Edge case:** No

### getRepoRoot — falls back to static resolution when tenantId absent

- **Verifies:** AC2
- **Precondition:** Session `{}` (no tenantId); `process.env.CLAUDE_REPO_PATH = '/repos/mine'`
- **Action:** `const root = getRepoRoot(makeReq({ session: {} })); assert.ok(root.includes('repos/mine'))`
- **Expected result:** Returns the static fallback path — not a tenant-namespaced path
- **Edge case:** Also test with no env vars set — should fall back to `path.resolve('.')` or equivalent

### slugifyTenantId — lowercases and replaces non-alphanumeric with hyphens

- **Verifies:** AC4
- **Precondition:** Input `'My Org'`
- **Action:** `assert.strictEqual(slugifyTenantId('My Org'), 'my-org')`
- **Expected result:** `'my-org'`
- **Edge case:** No

### slugifyTenantId — strips path separators

- **Verifies:** AC4
- **Precondition:** Input `'org/evil'` and `'org\\evil'`
- **Action:** Assert neither `/` nor `\` appear in output
- **Expected result:** Safe slug with no path separators
- **Edge case:** Yes — path traversal related

### slugifyTenantId — handles URL-encoded characters (does not decode)

- **Verifies:** AC4
- **Precondition:** Input `'org%2fevil'` (URL-encoded `/`)
- **Action:** `const slug = slugifyTenantId('org%2fevil'); assert.ok(!slug.includes('/'));`
- **Expected result:** Output contains no `/` — the `%` character replaced with `-`, not decoded
- **Edge case:** Yes — encoding safety

### slugifyTenantId — truncates to 48 characters

- **Verifies:** AC4
- **Precondition:** Input: `'a'.repeat(100)` (100-char string)
- **Action:** `assert.ok(slugifyTenantId('a'.repeat(100)).length <= 48)`
- **Expected result:** Slug length ≤ 48
- **Edge case:** Yes — length cap

### slugifyTenantId — strips leading and trailing hyphens

- **Verifies:** AC4
- **Precondition:** Input `'-org-a-'`
- **Action:** `const slug = slugifyTenantId('-org-a-'); assert.ok(!slug.startsWith('-')); assert.ok(!slug.endsWith('-'));`
- **Expected result:** No leading or trailing hyphens
- **Edge case:** No

## Integration Tests

### Path-traversal guard fires — HTTP 400 returned and no file written

- **Verifies:** AC3
- **Components involved:** Route handler that writes artefact files, `getRepoRoot`, path-traversal guard
- **Precondition:** `WUCE_TENANT_ROOT_BASE` = a temp dir; request with a crafted featureSlug `'../../etc/passwd'`
- **Action:** Call the artefact-writing route handler with the crafted featureSlug
- **Expected result:** (1) HTTP 400 response; (2) No file exists at `path.resolve(tmpDir, '../../etc/passwd')` — assert `!fs.existsSync(escapedPath)`
- **Edge case:** Yes — security-critical case; both assertions are mandatory per CLAUDE.md ougl standard

### Session written to tenant-namespaced path

- **Verifies:** AC5
- **Components involved:** Session-store adapter, `SESSION_STORE_PATH` env var
- **Precondition:** `process.env.SESSION_STORE_PATH = tmpDir`; session `{ tenantId: 'my-org', sessionId: 'sess-123', accessToken: 'tok', userId: '1', login: 'alice' }`
- **Action:** Call session-store write function
- **Expected result:** File exists at `path.join(tmpDir, 'my-org', 'sess-123.json')`; file does not exist at `path.join(tmpDir, 'sess-123.json')` (non-namespaced path)
- **Edge case:** No

### Session read from tenant-namespaced path only — cross-namespace isolation

- **Verifies:** AC6
- **Components involved:** Session-store adapter
- **Precondition:** File at `tmpDir/org-a/sess-1.json`; file at `tmpDir/org-b/sess-1.json` with different content
- **Action:** Read session for `{ tenantId: 'org-a', sessionId: 'sess-1' }`
- **Expected result:** Returns content of `org-a/sess-1.json`; does not return `org-b/sess-1.json` content
- **Edge case:** Yes — namespace isolation

### featureSlug collision guard — same slug, different tenants, written to separate directories

- **Verifies:** AC7
- **Components involved:** `getRepoRoot`, artefact-path construction
- **Precondition:** Two calls with `tenantId='org-a'` and `tenantId='org-b'`, both with `featureSlug='my-feature'`
- **Action:** Write a file using each tenant's repoRoot + featureSlug
- **Expected result:** Files written to `WUCE_TENANT_ROOT_BASE/org-a/.../my-feature/...` and `WUCE_TENANT_ROOT_BASE/org-b/.../my-feature/...` — distinct directories; reading org-a's path returns org-a's file, not org-b's
- **Edge case:** No

### accessToken strip-before-write preserved with namespaced session path

- **Verifies:** AC8
- **Components involved:** Session-store adapter
- **Precondition:** Session includes `accessToken: 'secret'`; written to tenant-namespaced path
- **Action:** Read file from disk after write; parse as JSON
- **Expected result:** `parsedFile.accessToken === undefined` (key stripped before write; not present in persisted file)
- **Edge case:** No

## NFR Tests

### getRepoRoot is synchronous — completes in under 1ms

- **NFR addressed:** Performance — "getRepoRoot must return in under 1ms (synchronous path resolution, no I/O)"
- **Measurement method:** `assert.ok(!(getRepoRoot(req) instanceof Promise))` — assert synchronous return; time 1000 calls and assert median < 1ms
- **Pass threshold:** Synchronous; median < 1ms for 1000 iterations
- **Tool:** Plain Node.js `process.hrtime()` in the test

### Session writes are non-blocking (async write pattern)

- **NFR addressed:** Performance — "session file writes must not block the event loop"
- **Measurement method:** Assert session write function returns a Promise (is async/awaited); code inspection at DoR confirms no `fs.writeFileSync` in session-store write path
- **Pass threshold:** Session store write is async
- **Tool:** Code inspection + `assert.ok(writeResult instanceof Promise)`

### Raw tenantId not logged on path-traversal guard failure

- **NFR addressed:** Security — "do not log the raw tenant-derived path value in production" (CLAUDE.md ougl)
- **Measurement method:** Capture log output during AC3 test (path-traversal guard fires); assert log does not contain the raw `tenantId` or `featureSlug` value
- **Pass threshold:** Log output contains no raw path data from the crafted input
- **Tool:** Log spy (override `console.error` / logger in test scope)
