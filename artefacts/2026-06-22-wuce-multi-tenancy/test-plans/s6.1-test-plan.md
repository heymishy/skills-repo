## Unit / Integration Tests

(All tests below are integration-level in the sense that they exercise the real, unmodified call chain from route handler down to the mocked HTTP boundary — no unit isolates a single function in this story, because the defect is in wiring between functions, not in any one function's logic. This matches the story's own framing: activation, not new logic.)

### AC1 — Non-streaming: two tenants, identical prompt → two different captured system texts

- **Verifies:** AC1
- **Precondition:** `routes.setSkillTurnExecutorAdapter` left wired to the real module executor (not stubbed); `https.request` mocked to capture the outbound body and return a canned 200 response; `ANTHROPIC_API_KEY` set to a fake value; `SKILL_EXECUTOR_PROVIDER` unset/`anthropic`
- **Action:** Call `handlePostTurnHtml(req, res)` twice with `req.session.tenantId = 'org-a'` then `'org-b'`, same `session.systemPrompt` and same `sessionId` (session re-fetched fresh each call to avoid turn-history cross-talk)
- **Expected result:** The two captured `system[0].text` values are not equal (`assert.notStrictEqual`)
- **Edge case:** Yes — this is the primary safety case Decision 8 exists to close

### AC2 — Non-streaming: cache-scope line format

- **Verifies:** AC2
- **Precondition:** Same harness as AC1; `req.session.tenantId = 'my-org'`, `sessionId` known
- **Action:** Inspect captured `system[0].text.split('\n')[0]`
- **Expected result:** Exactly `<!-- cache-scope: my-org-<sessionId> -->`
- **Edge case:** No

### AC3 — Streaming: two tenants, identical prompt → two different captured system texts

- **Verifies:** AC3
- **Precondition:** Same as AC1 but via `handlePostTurnStreamHtml(req, res)`, with `https.request` mocked to emit a minimal valid SSE `message_start`/`content_block_delta`/`message_stop` sequence so the handler completes without hanging
- **Action:** Call twice with `req.session.tenantId = 'org-a'` then `'org-b'`, same prompt/sessionId
- **Expected result:** Captured `system[0].text` values differ between the two calls
- **Edge case:** Yes — proves the streaming path was not left behind

### AC4a — Non-streaming: no tenantId → prompt unchanged

- **Verifies:** AC4
- **Precondition:** `req.session.tenantId` absent
- **Action:** Call `handlePostTurnHtml(req, res)`; capture `system[0].text`
- **Expected result:** `system[0].text === systemPrompt` (no `cache-scope` line)
- **Edge case:** Also test `tenantId: null`

### AC4b — Streaming: no tenantId → prompt unchanged

- **Verifies:** AC4
- **Precondition:** `req.session.tenantId` absent
- **Action:** Call `handlePostTurnStreamHtml(req, res)`; capture `system[0].text`
- **Expected result:** `system[0].text === systemPrompt`
- **Edge case:** No

### AC6 — Regression: existing suites pass unmodified

- **Verifies:** AC6
- **Precondition:** Implementation complete

- **Edge case:** No
