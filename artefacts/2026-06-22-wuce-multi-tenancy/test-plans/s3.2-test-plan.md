## Integration Tests

> **TDD note:** Tests are implemented by `scripts/smoke-test-redis.js` (already exists). They skip gracefully when env vars are absent (`SKIP: UPSTASH_REDIS_REST_URL not set`). Failing state = env vars not set.

### T1 — SET session key with TTL writes successfully

- **Verifies:** AC2 (write leg)
- **Components involved:** smoke-test-redis.js → @upstash/redis client → Upstash REST API
- **Precondition:** `UPSTASH_REDIS_REST_URL` and `UPSTASH_REDIS_REST_TOKEN` set
- **Action:** `UPSTASH_REDIS_REST_URL=<url> UPSTASH_REDIS_REST_TOKEN=<token> node scripts/smoke-test-redis.js`
- **Expected result:** stdout contains `SET session:smoke-test ... OK`

### T2 — GET returns the written value

- **Verifies:** AC2 (read leg)
- **Components involved:** smoke-test-redis.js → client.get()
- **Precondition:** T1 completed
- **Action:** Continues in same script run
- **Expected result:** stdout contains `GET session:smoke-test ... OK`; raw value is non-null

### T3 — accessToken absent from stored value; login present

- **Verifies:** AC3
- **Components involved:** smoke-test-redis.js → stored value parsed as JSON
- **Precondition:** T2 completed
- **Action:** Checks `stored.accessToken === undefined` and `stored.login === 'smoke-user'`
- **Expected result:** stdout contains `accessToken absent, login present ... OK`

### T4 — TTL is within [86390, 86400] seconds

- **Verifies:** AC4
- **Components involved:** smoke-test-redis.js → client.ttl()
- **Precondition:** T1 completed (write with ex: 86400)
- **Action:** `client.ttl('session:smoke-test')` immediately after write
- **Expected result:** stdout contains `TTL=<n>s (within [86390, 86400]) ... OK`; n ≥ 86390 and n ≤ 86400

### T5 — DEL removes the key

- **Verifies:** AC2 (cleanup leg)
- **Components involved:** smoke-test-redis.js → client.del() + client.get()
- **Precondition:** T1 completed
- **Action:** Delete key; follow-up GET returns null
- **Expected result:** stdout contains `DEL session:smoke-test ... OK`; script exits 0; stdout ends with `Upstash smoke test PASSED`
