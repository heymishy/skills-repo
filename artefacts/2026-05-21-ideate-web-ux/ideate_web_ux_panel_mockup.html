# Assumption Inventory — ideate-web-ux
## Desirability
D-1 [High · Guess] Operators will actively confirm or flag assumption cards mid-session rather than ignoring them. This is the central behavioural assumption of the Cluster 2 value proposition — if operators stay in chat-reading flow during generation, the cards accumulate unseen.
D-2 [Medium · Inference] The context manifest panel provides enough signal for operators to catch missing context before the first lens runs. Depends on manifest completeness and operator familiarity with expected context sources.
