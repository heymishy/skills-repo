## What will be built

1. `scripts/platform-init.js` — Node.js CLI script (built-ins only):
   - Copies `.github/skills/`, `.github/templates/`, and core `scripts/` files from platform source to target repo
   - Creates or updates `.github/copilot-instructions.md` with standard platform instructions
   - Skips existing files by default; supports `--force` flag to overwrite
   - Outputs installation summary listing installed and skipped files
   - Exits non-zero with human-readable error message on failure

2. `scripts/platform-fetch.js` — Node.js CLI script (built-ins only):
   - Fetches current HEAD skill files from platform's master branch (local path or git archive)
   - Copies updated files to repo
   - Records fetch timestamp in `workspace/platform-fetch-log.json` as `{ "fetchedAt": "<ISO-8601>", "source": "<identifier>" }`
   - Exits non-zero with human-readable error message on failure

3. `package.json` — adds two script entries:
   - `"platform:init": "node scripts/platform-init.js"`
   - `"platform:fetch": "node scripts/platform-fetch.js"`

## What will NOT be built

- `platform:pin` and `platform:verify` commands (i1.3 scope)
- Automated conflict resolution (--force flag only)
- Alternative package manager support
- `dependencies` section additions to `package.json`
