## Scenarios

### Scenario 1 — Codebase without pipeline artefacts: routes to Entry B with full path (AC1)

**How to verify (SKILL.md inspection):**
Find the Entry B routing block.

**Expected outcome (check all):**
- `[ ]` The routing block is labelled "Entry B — code-first brownfield" (or equivalent canonical phrase)
- `[ ]` `/reverse-engineer` is named and described as extracting business rules and design from the existing codebase
- `[ ]` The full suggested path is shown: `/reverse-engineer` → `/discovery` → `/benefit-metric` → `/definition` (all four skills in sequence)

**Pass:** All three checkboxes satisfied.
**Fail:** Any checkbox unchecked, or the full four-skill path is incomplete.

### Scenario 2 — Detection signal named; config-only files do not trigger Entry B (AC2 + NFR)

**How to verify (SKILL.md inspection):**
Read the detection condition for Entry B.

**Expected outcome (check all):**
- `[ ]` The detection condition specifies application logic file extensions (at minimum: .js, .ts, .py, .java, .rb, .go, .cs)
- `[ ]` Config-only files (.yml, .json, Dockerfile) are either explicitly excluded or the detection description is precise enough that a repo with only config files would NOT match
- `[ ]` The classification output instructs naming the specific directories found (e.g. "Found: src/ with application files")

**Pass:** All three checkboxes satisfied.
**Fail:** Any checkbox unchecked, or detection criteria are too broad (any file in src/ matches).

### Scenario 3 — Repo with both code AND stories: Entry A fires, not Entry B (AC3)

**How to verify (SKILL.md inspection):**
Read the routing priority section.

**Expected outcome (check all):**
- `[ ]` Entry B detection requires absence of any pipeline artefacts (including story artefacts)
- `[ ]` The routing priority is explicit: Entry A (story artefacts present) takes precedence over Entry B
- `[ ]` If running live against a repo with both `artefacts/[feature]/stories/` AND `src/`, the response names Entry A, not Entry B

**Pass:** All checkboxes satisfied.
**Fail:** Entry B fires even when story artefacts are present.
