## AC6 — outcome-orientation diagnostic

AC6 requires each standards requirement in each `core.md` to pass the diagnostic: "if the model behaviour improved past this behaviour, would this requirement still be correct?" This is a human authoring review at P1.7 DoD time, not an executable automated test. The verification script captures this as a manual confirmation step. The `reviewedBy` and `reviewDate` fields in each standards file are the observable artefact of the review.
