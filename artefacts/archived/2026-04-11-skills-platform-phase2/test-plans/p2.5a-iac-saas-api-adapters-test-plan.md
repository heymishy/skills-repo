## Unit Tests

### IaC adapter: execute() is callable with (surface, context) and returns a result object

- **Verifies:** AC1
- **Precondition:** `src/surface-adapter/adapters/iac.js` (or equivalent) is implemented
- **Action:** Call `execute('iac', minimalContext)`
- **Expected result:** Returns an object with fields: `status` (string), `findings` (array), `trace` (object)
- **Edge case:** No

### IaC adapter: result.trace contains adapterName and adapterVersion fields

- **Verifies:** AC1
- **Precondition:** As above
- **Action:** Inspect `result.trace`
- **Expected result:** `result.trace.adapterName` is `"iac"` (or similar non-null identifier); `result.trace.adapterVersion` is a non-empty string
- **Edge case:** No

### IaC adapter: execute() with no POLICY.md file returns an error result (not a thrown exception)
