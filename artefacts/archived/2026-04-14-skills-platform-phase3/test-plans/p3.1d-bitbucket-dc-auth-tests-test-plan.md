### U2-A — AC2 (Option A): Workflow triggers on `schedule`, not `pull_request` or `push`

**File:** `tests/check-bitbucket-dc-auth.test.js`
**Applicable when:** Option A is chosen.

**Assertion:** Read `.github/workflows/bitbucket-dc-auth.yml`. Parse YAML `on:` key. Assert `schedule` is present. Assert neither `pull_request` nor `push` appears under `on:`.

**Expected (FAIL before impl):** File absent, or `on:` contains `pull_request` or `push`.
**Expected (PASS after impl):** `on.schedule` present; `pull_request` and `push` absent from `on:` block.

---

### U3-A — AC4 (Option A): Auth tests have no permanent `// PREREQ-DOCKER` skip annotation

**File:** `tests/check-bitbucket-dc-auth.test.js`
**Applicable when:** Option A is chosen.

**Assertion:** Read `tests/check-bitbucket-dc.js`. Assert the four auth test sections (`app-password`, `OAuth`, `SSH key`, `PAT`) no longer contain an active `test.skip` or `// PREREQ-DOCKER` permanent-skip pattern.

**Expected (FAIL before impl):** One or more of the four tests remains annotated with `test.skip` and `// PREREQ-DOCKER` without any alternative resolution.
**Expected (PASS after impl):** None of the four tests has a permanent skip. They either run unconditionally or are conditionally skipped via `process.env.SKIP_DOCKER_TESTS` (not a permanent annotation).

### U5-B — AC3 (Option B): Each auth test entry in smoke-tests.md has all 5 required fields

**File:** `tests/check-bitbucket-dc-auth.test.js`
**Applicable when:** Option B is chosen.

**Assertion:** For each of the four auth test entries, assert all five required fields are present:
1. Test name
2. Last-run date (ISO 8601 format — matches `/\d{4}-\d{2}-\d{2}/`)
3. Result (`pass` or `fail` — case-insensitive)
4. Runner identity (non-empty string in the runner field)
5. Docker dependency rationale note (contains "Docker" or "docker")

**Expected (FAIL before impl):** One or more entries missing a field (e.g. no ISO date, empty runner field).
**Expected (PASS after impl):** All 4 entries × 5 fields = 20 field assertions pass.
