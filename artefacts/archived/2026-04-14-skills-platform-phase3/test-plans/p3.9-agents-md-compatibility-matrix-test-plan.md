## Unit Tests

### compatibility-matrix-file-exists

- **Verifies:** AC1
- **Precondition:** Story is implemented
- **Action:** Check that `docs/agent-compatibility-matrix.md` exists at the repo root docs directory
- **Expected result:** File exists
- **Edge case:** No

### compatibility-matrix-has-required-tool-rows

- **Verifies:** AC1
- **Precondition:** `docs/agent-compatibility-matrix.md` exists
- **Action:** Read the file; parse the markdown table; extract tool name rows; verify all four required tools are present: "GitHub Copilot", "Cursor", "Claude Code", and at least one of ["Amazon Q Developer", "JetBrains AI"] or any named enterprise-equivalent
- **Expected result:** Table contains exactly 4+ rows (one per tool); required three are present; fourth is a named enterprise tool
- **Edge case:** No

### compatibility-matrix-has-required-columns

- **Verifies:** AC1
- **Precondition:** `docs/agent-compatibility-matrix.md` exists
- **Action:** Parse the markdown table header row; verify these 6 columns are present: tool name, instruction surface resolution, system prompt delivery on task start, DoR handoff format compatibility, trace generation, known gaps
- **Expected result:** All six columns present (exact column header text may vary slightly but semantics must match)
- **Edge case:** No

### partial-and-no-entries-have-actionable-gap-descriptions

- **Verifies:** AC2
- **Precondition:** `docs/agent-compatibility-matrix.md` exists
- **Action:** Parse all cells in the known-gaps column for rows where any other column contains "partial" or "no"; check that none of these cells contain only "unknown" or are empty
- **Expected result:** Every "partial" or "no" row has a non-empty, non-"unknown" gap description in the known-gaps column; descriptions must be more than 5 words and name a specific mechanism (e.g. ".cursorrules" not just "incompatible")
- **Edge case:** No

### phase4-backlog-file-exists-with-required-content

- **Verifies:** AC3
- **Precondition:** Story is implemented
- **Action:** Check that `workspace/phase4-backlog-agent-live-delivery-run.md` exists; read the file; verify it contains (a) a story title, (b) the text "ASSUMPTION-04" and the phrase "operational confirmation" or equivalent, (c) a prerequisites section naming the tools, story, and squad requirements
- **Expected result:** File exists; contains all three required content elements
- **Edge case:** No

### pr-diff-contains-only-two-expected-files

- **Verifies:** AC4
- **Precondition:** PR is open
- **Action:** Inspect the PR diff file list
- **Expected result:** Exactly two files changed: `docs/agent-compatibility-matrix.md` and `workspace/phase4-backlog-agent-live-delivery-run.md`; no SKILL.md, no scripts, no CI files
- **Edge case:** No
