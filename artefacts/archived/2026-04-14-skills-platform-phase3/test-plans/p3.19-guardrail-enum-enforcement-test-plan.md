### M5 — AC3: Enum values are exactly correct in both files

**Command:**
```powershell
# Check status values in both files
foreach ($file in @(".github/skills/definition-of-done/SKILL.md", ".github/skills/definition-of-ready/SKILL.md")) {
    $content = Get-Content $file -Raw
    $hasStatus = $content -match "met.*not-met.*na.*excepted.*not-assessed"
    $hasCategory = $content -match "mandatory-constraint.*adr.*nfr.*compliance-framework.*pattern.*anti-pattern"
    Write-Host "$file — Status: $hasStatus, Category: $hasCategory"
}
```

**Expected (FAIL before impl):** Both return False or file not found.
**Expected (PASS after impl):** Both files return `Status: True, Category: True`.
