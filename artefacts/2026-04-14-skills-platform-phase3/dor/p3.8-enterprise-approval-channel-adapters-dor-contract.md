## What will be built

1. **`src/approval-channel/adapters/teams-adapter.js`** — new Node.js CommonJS module. Implements the approval-channel adapter interface for Microsoft Teams. Parses Adaptive Card webhook payload, verifies HMAC signature using a signing secret from environment variable, extracts the story ID, and calls `process-dor-approval.js`. Returns HTTP 4xx/5xx on failure. No hardcoded credentials.

2. **`src/approval-channel/adapters/jira-adapter.js`** — new Node.js CommonJS module. Implements the approval-channel adapter interface for Jira. Supports two trigger variants: workflow transition to "Approved" state or `/approve-dor` comment on the linked issue. Verifies service account token from environment variable. Calls `process-dor-approval.js` with the story ID. Returns HTTP 4xx/5xx on failure. No hardcoded credentials.

---

## What will NOT be built

- No ServiceNow adapter (Phase 4)
- No Slack adapter (Phase 4)
- No email-based approval
- No modification to the `src/approval-channel/` interface contract — only adapter implementation
