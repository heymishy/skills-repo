## What will be built

1. **`tests/check-definition-skill.js`** — import path fix only. The `require()` call that imports from `improvement-agent/` must be updated to use the correct relative path pointing into `src/`. No other changes to this file.

2. **`src/improvement-agent/watermark-gate.js`** (confirm actual path) — add a named constant `FLOOR_PASS_RATE` with value `0.70`. Add enforcement logic: if `passRate < FLOOR_PASS_RATE` at baseline-creation time, the gate exits 1 with an explanatory message. The constant name `FLOOR_PASS_RATE` must appear literally in the source (AC2 test asserts on this string).

3. **`src/improvement-agent/failure-detector.js`** (confirm actual path, may be named differently) — update counter logic to track additions and removals separately. Current behaviour: a net-zero change (add 1, remove 1) exits 0. New behaviour: if `removedCount > 0`, exit 1 regardless of `addedCount`. If `removedCount === 0` and `addedCount > 0`, exit 0. Counts must be reported separately in stdout. Does not retroactively re-evaluate previously accepted proposals (ADR-008 constraint).
