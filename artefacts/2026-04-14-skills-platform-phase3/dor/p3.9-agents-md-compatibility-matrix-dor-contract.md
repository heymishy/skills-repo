## What will be built

1. **`docs/agent-compatibility-matrix.md`** — new markdown document. Contains a compatibility matrix with one row per tool. Tools covered: GitHub Copilot (baseline), Cursor, Claude Code, and one additional enterprise-equivalent agent (Amazon Q Developer or JetBrains AI). Each row contains: tool name, instruction surface resolution (yes/no/partial), system prompt delivery on task start (yes/no/partial), DoR handoff format compatibility (yes/no/partial), trace generation (yes/no/not supported), and known gaps. Every "partial" or "no" entry has a specific, actionable gap description.
