## Overview

Phase 2 approval channel: `github-issue`. Approver posts `/approve-dor` as a comment on the linked GitHub Issue. A GitHub Actions workflow detects the comment and writes `dorStatus: "signed-off"`, `dorApprover` (GitHub username), and `dorChannel: "github-issue"` to the squad's `pipeline-state.json`. Channel targets are always read from `context.yml` â€” no hardcoded values.
