## Unit Tests

### H8-ext: DoR contract builder prompts for schemaDepends when upstream dependency declared

- **Verifies:** AC1
- **Precondition:** Story `tests/fixtures/minimal-story.md` modified to add `Upstream: p2.1-definition-skill-improvements` in Dependencies block
- **Action:** Run the DoR contract builder against this fixture; check output for presence of `schemaDepends` field prompt or placeholder
- **Expected result:** DoR contract contains `schemaDepends:` field entry (may be empty `[]` prompt); operator is prompted to declare schema fields before sign-off
- **Edge case:** No

### H8-ext hard block: schemaDepends field declared but absent from pipeline-state.schema.json

- **Verifies:** AC2
- **Precondition:** DoR contract with `schemaDepends: ["testOrphanField_p23_ac2"]`; `pipeline-state.schema.json` does not contain `testOrphanField_p23_ac2`
- **Action:** Run H8-ext check against this fixture combination
- **Expected result:** Check result is `FAIL` with message containing "H8-ext FAIL:" and naming `testOrphanField_p23_ac2` as the missing field; result instructs adding to schema or correcting declaration
- **Edge case:** No

## NFR Tests

### Consistency: H8-ext produces identical result for identical input regardless of session order

- **NFR addressed:** Consistency
- **Measurement method:** Run H8-ext check against the same fixture twice in separate invocations; compare PASS/FAIL result and message
- **Pass threshold:** Both runs produce the same PASS or FAIL result and the same message text
- **Tool:** Node.js test script

### Backward compatibility: D9 DoD template change does not break Phase 1 DoD artefacts

- **NFR addressed:** Backward compatibility
- **Measurement method:** Read a Phase 1 DoD artefact (e.g. `artefacts/2026-04-09-skills-platform-phase1/dod/p1.1-*.md`) through the DoD template parser; confirm it does not error on the new Verification method field being absent
- **Pass threshold:** Phase 1 DoD artefact parsed without error; no schema violation warning about the Verification method field
- **Tool:** Node.js test script (or `check-pipeline-artefact-paths.js`)
