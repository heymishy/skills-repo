## Epic: Fleet observability and non-engineer approval interface

**Discovery reference:** artefacts/2026-04-11-skills-platform-phase2/discovery.md
**Benefit-metric reference:** artefacts/2026-04-11-skills-platform-phase2/benefit-metric.md
**Slicing strategy:** Risk-first

## Goal

When this epic is complete, the platform maintainer can see the live pipeline state of all registered squads in the fleet panel of pipeline-viz without manually querying individual repos or editing shared files. Non-engineer stakeholders (PMs, risk leads, compliance leads) can complete a DoR sign-off action from Jira, Confluence, Slack, or Teams without opening VS Code. The approval intent is recorded in the non-engineer interface and `pipeline-state.json` is updated with the canonical `dorStatus: signed-off` evidence field.

## Out of Scope

- Federated cross-organisation fleet aggregation — Phase 2 aggregation runs within a single platform repo. Cross-org or cross-GitHub-org federation is Phase 3.
- Full automated integration between non-engineer approval interfaces and `pipeline-state.json` — manual processing of the interface record is acceptable in Phase 2. Full webhook-to-state-write automation is Phase 3.
- Real-time fleet state updates (sub-minute refresh). Fleet-state.json is updated on CI job trigger (scheduled or push), not continuously.
- Microsoft Teams bot, Slack app, or Jira webhook development requiring OAuth app registration — Phase 2 uses native messaging interfaces (reactions, replies, comments).
- Multi-approver workflows or approval quorum logic.

## Human Oversight Level

**Oversight:** High
**Rationale:** Fleet registry introduces a new data file pattern and schema (`fleet/squads/{id}.json`, `fleet-state.json`) with no Phase 1 precedent. Persona routing modifies how approval actions flow into `pipeline-state.json` — the `dorStatus` evidence field is a hard dependency for downstream DoR and DoD gates. Errors in either story could corrupt pipeline state for real squads. Human review required before any inner loop execution.
