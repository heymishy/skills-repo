# Manual Policy Floor

**Surface:** manual
**lastReviewedBy:** TBD
**lastReviewedDate:** TBD

These floors are binary. A delivery either meets them or it does not. No domain or squad override may relax these floors.

## Governance Criteria

- MUST obtain change request approval before the manual change is executed so that all manual changes are authorised
- MUST attach evidence of completion for each checklist item so that the manual process outcome is verifiable
- MUST confirm a rollback plan is in place before executing a manual change so that unintended effects can be reversed
