# Epic: Operational Automation — Phase 1

**Epic slug:** operational-automation-phase-1
**Feature slug:** regulatory-reporting-pipeline-automation
**Status:** In scope for Phase 1

## Goal
Deliver the regulated reporting path that automates extraction, pre-population, review, audit logging, and submission confirmation without activating the normalisation transformation layer.

## Stories
1. Extract and pre-populate the RBNZ monthly return.
2. Implement immutable audit trail infrastructure.
3. Deliver the analyst review and approval workflow.
4. Log submission confirmations and dispatch to the RBNZ/FMA gateways.
5. Schedule monthly extraction and alert on deadline risk.

## Governance note
The normalisation layer remains outside scope of this epic and is gated by the Compliance Officer production activation clearance path described in the discovery brief.
